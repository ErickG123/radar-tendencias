title: ADR 002 - Stack e Padronização do Frontend
---
## Status
Aceito

## Decisão
Utilizar Angular com organização feature-based, restrito ao uso de Standalone Components e Signals. A interface utilizará PrimeNG. O Dark Mode exige customização via SCSS para garantir que os textos e cabeçalhos mudem de cor dinamicamente, prevenindo textos invisíveis.
