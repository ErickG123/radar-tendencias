## O que é a aplicação?
O Radar de Tendências Geek é uma plataforma de inteligência de mercado e social listening focada na cultura pop. A aplicação coleta, processa e analisa grandes volumes de dados não estruturados utilizando NLP para traduzir o volume de interações em um indicador consolidado de relevância (Hype Score).

## Para que serve?
* Antecipação de Demanda
* Otimização de Estoque
* Gestão de Hype
* Alertas Estratégicos

## Funcionalidades Principais
1. Ingestão Multicanal de Dados
2. Motor de Processamento (NLP)
3. Dashboard Analítico de Tendências
4. Sistema de Alertas e Notificações

## Stack Tecnológica Base
* Frontend: Angular com Standalone Components, Signals e PrimeNG.
* Backend: APIs em .NET com Dapper.
* Bancos: SQL Server e Elasticsearch.
* Mensageria: RabbitMQ ou Kafka.
